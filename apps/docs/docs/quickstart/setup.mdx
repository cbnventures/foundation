# Setup

Foundation Nova supports two starting paths. Use the runner for a one-off, install globally, or install into project for configs you can commit.

## Requirements

- Node.js <abbr title="Long-Term Support">LTS</abbr>
- Node Package Manager (npm)

:::tip Quick Check
Run this command below to check the installed versions of your Node.js, package managers, operating system, installed browsers, and other dev tools directly:

```bash
npx --yes @cbnventures/foundation-nova@latest version --all
```
:::

## Choose Your Path

Choose **Run once (npx)** to try Nova or make a one-off change, **Install globally**, or **Install into project** to use preset configs and run commands any time.

<Tabs groupId="setup-mode">
  <TabItem value="once" label="Run once (npx)" default>

    View the built-in help from the command below, or see **[CLI](/docs/cli/)**.

    ```bash
    npx --yes @cbnventures/foundation-nova@latest
    ```

  </TabItem>
  <TabItem value="global" label="Install globally">

    View the built-in help from the command below, or see **[CLI](/docs/cli/)**.

    ```bash
    npm i -g @cbnventures/foundation-nova@latest
    ```

    You can use the shorthand command for the **[CLI](/docs/cli/)** after installation:

    ```bash
    nova
    ```

  </TabItem>
  <TabItem value="local" label="Install into project">

    First, Install Nova.

    ```bash
    npm i -D @cbnventures/foundation-nova@latest
    ```

    Second, configure **TypeScript** though the `tsconfig.json` file. See **[TSConfig best practices](/docs/configuration/tsconfig/best-practices)**.

    ```json
    {
      "extends": [
      "./src/presets/tsconfig/dx-essentials.json",
      "./src/presets/tsconfig/dx-strict.json"
      ]
    }
    ```

    Third, configure **ESLint** through the `eslint.config.mts` file. See **[ESLint best practices](/docs/configuration/eslint/best-practices)**.

    ```javascript
    import { dxCodeStyle, scopeIgnores } from '@cbnventures/foundation-nova/eslint';

    export default [
      ...scopeIgnores,
      ...dxCodeStyle,
    ];
    ```

    Optionally, you can use the shorthand command for the **[CLI](/docs/cli/)**:

    ```bash
    nova
    ```

  </TabItem>
</Tabs>
